Norenown Gaming Web Store

      Nowadays, gaming has emerged as a prominent form of entertainment for individuals of almost every age. However, not everyone has the time to visit a physical game store to purchase games or related items. This is where the Norenown Web Store comes to the rescue. This website offers users the convenience of selecting and buying games, as well as game-related items like keychains, pillows, and in-game merchandise, all online, and at particularly reasonable prices.The project is developed using PHP for the backend, employing the MVC model for efficient data flow, and incorporating HTML/CSS for the frontend. This combination of technologies ensures a seamless and user-friendly experience for online shoppers interested in the gaming niche.

Hybrid Class Management Application

      This desktop app provides a handful of tools to manage your online classes. For teachers, we offer the ability to create online classes for their students to join, enabling them to easily assign homework, exams, and share documents. For students, we provide an advanced notification feature to remind them as their deadlines approach. The app is developed entirely in C#, leveraging Visual Studio and the Krypton UI library for a seamless and visually appealing user experience.
